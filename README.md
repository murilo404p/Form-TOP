## 💁‍♂️ Sobre o Projeto

Este projeto consiste em uma página de cadastro (**Sign-up Form**) desenvolvida como parte do currículo do **The Odin Project**.

A aplicação reproduz um layout moderno de formulário de registro, colocando em prática conceitos de HTML semântico, estilização com Tailwind CSS, posicionamento de elementos e validação de formulários utilizando JavaScript.

Durante o desenvolvimento, foram explorados conceitos fundamentais de estruturação de páginas, manipulação do DOM e criação de interfaces responsivas inspiradas em um design real.

---

## 📅 Desenvolvimento

Este projeto foi desenvolvido durante meus estudos de HTML, CSS e JavaScript no **The Odin Project**, com o objetivo de praticar a construção de formulários, organização semântica do código e manipulação de eventos.

Além da reprodução do layout proposto, a aplicação realiza validações de campos obrigatórios e verifica se as senhas informadas pelo usuário são compatíveis, proporcionando uma melhor experiência de uso.

---

## ✨ Funcionalidades

- 📝 Formulário completo de cadastro
- 🔒 Validação de confirmação de senha
- ⚠️ Mensagens de erro para entradas inválidas
- 📧 Validação de e-mail utilizando HTML5
- 📱 Interface responsiva
- 🎨 Estilização moderna utilizando Tailwind CSS
- ⚡ Ambiente de desenvolvimento configurado com Vite
- 🧩 Estrutura HTML semântica
